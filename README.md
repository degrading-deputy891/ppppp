## 📖 About this application

The ppppp application helps you sort, organize, and track your digital files. Many people gather large collections of documents, images, and folders on their computers. Over time, these files become hard to manage. This tool simplifies that process. It monitors specific folders on your Windows computer and keeps your data clean. You save time because you spend less effort searching for lost items.

## ⚙️ System requirements

This application runs on most modern computers. Please ensure your computer meets these basic needs before you install:

* Windows 10 or Windows 11.
* At least 200 MB of free storage space.
* An active internet connection for initial setup.
* 4 GB of RAM or higher.

If your computer meets these standards, the program will run without issues.

## 🛠️ Configuration guide

After you open the app, you will see a main window. The first step involves picking the folders you want the app to watch. Click the Add Folder button. A window opens that allows you to browse your computer files. Find the folder you want to organize and click Select.

The app uses default rules to sort your files once you add a folder. For example, it moves images into a specific picture folder and documents into a folder for writing. You can change these rules in the Settings menu. Click the gear icon to open the settings view. Here, you see a list of file types and their destinations. You can move these items by dragging the folders to new locations. 

## 💡 Using the software

The program runs in the background while you work on your computer. You see a small icon in the taskbar near your clock. If you want to check the status of your files, click this icon to bring the main window to the front.

If you add a new file to a watched folder, the app detects the change within seconds. It moves the file to the correct category based on your settings. You do not need to perform any manual tasks once you set your preferences. The software handles the movement of data
